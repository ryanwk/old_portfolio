About



This was my first team project, we were tasked with building a full-stack E-Commerce Website. We built this site in 3 days, with only 8 weeks of programming experience. Each team member carried out specific roles during our 3 days of development.

Click the 'E-Commerce' button to check out our site.

Product viewing, shopping cart actions, user authentication, and storing and retrieving product interactions were performed with JavaScript. We used AJAX to send api requests to the back end. The back end was built with MongoDB and Express. We used stripe to accept credit card payments.

If you are interested in viewing more of the technical specifications of the site, a detailed description of how it was built, or viewing the wireframes and user stories that went into planning, then check out my Github repository. You can find a link to my repository down below!
