About

Stronger was my first attempt at building a full-stack application. I built this application in about a week while at General Assembly. The front-end needs some love and I'm in the process of building out a more comprehensive version with Angular and Ionic. I built this web app because I was unhappy with current apps on the market so I decided to build my own. My goal was to create an app to fit my specific needs while working out. Those needs are a platform to record weekly progress on my lifts while incrementally adding weight to the bar from weeek to week.

Click the 'Workout' button to check out my Fitness Tracking app.

The game logic, handling user authentication, and storing and retrieving game data was performed with JavaScript. I used AJAX to send api requests to the back end. The back end was built using Ruby and Rails. I set up the database using PostresSQL. The database stores exercise data in a way to allow users to view the date they updated an exercise, the weight the lifted, and the name of the exercise performed. The exercise data is displayed on the front end using JavaScript. The user interface was designed using HTML, CSS, Bootstrap and Handlebars.
