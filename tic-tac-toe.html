About

Tic Tac Toe was my first attempt at building a front-end application. I built this game in 5 days with only three weeks of programming experience. I was blown away at how much thought, logic, and attention to detail went into building such a seemingly simple game. I have a new-found appreciation for the amount of effort that goes into building simple games like Tic Tac Toe. This game marks the beginning of my journey into web development. I went on to build various other full-stack web applications. Looking back, it's humorous how challenging this game was. As I've grown as a developer I've built full-stack applications with vanilla javascript, AJAX, Express.js, Node.js, and back-end technologies such as Ruby on Rails and MongoDB. Recently, I have discovered the power of front-end frameworks and I'm currently working towards revamping some old projects with Angular and eventually Ionic to support mobile development.

Click the 'Play' button to check out my Tic Tac Toe game.

The game logic, handling user authentication, and storing and retrieving game data was performed with JavaScript. I used AJAX to send api requests to the back end. The back end was provided to me by General Assembly.

If you are interested in viewing more of the technical specifications of the game, a detailed description of how to play, or viewing the wireframes and user stories that went into planning, then check out my Github repository. You can find a link to my repository down below!
